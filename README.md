# 随机图片插件

AstrBot 随机图片插件，发送指令后随机发送一张图片。

## 功能特性

- 发送 `/随机拍切姿势` 随机发送一张图片
- 支持多种图片格式：png, jpg, jpeg, gif, webp, bmp

## 使用方法

1. 将图片放入插件目录下的 `data` 文件夹
2. 在群聊或私聊中发送 `/随机拍切姿势`
3. 机器人会随机发送一张图片

## 指令列表

| 指令 | 说明 |
|------|------|
| `/随机拍切姿势` | 随机发送一张图片 |

## 文件结构

```
astrbot_plugin_weibo/
├── main.py          # 插件主文件
├── data/            # 图片存放目录（将图片放这里）
│   ├── 图片1.jpg
│   └── 图片2.png
├── requirements.txt # 依赖（无额外依赖）
└── metadata.yaml    # 插件元数据
```

## 依赖

无额外依赖，仅使用 AstrBot 内置模块。
